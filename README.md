# CollabHub

A platform for developers to find teammates, post projects, and collaborate on hackathons.

## Tech Stack

- **Frontend:** React + Vite
- **Backend:** Node.js + Express + PostgreSQL

## Features

- 🔐 Auth (Register / Login / JWT)
- 🗂️ Post & Explore Projects
- 👥 Find Teammates
- 🏆 Hackathon Listings
- 💬 Real-time Chat

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
cp .env.example .env
npm run migrate
npm run dev
```
